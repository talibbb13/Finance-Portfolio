# Nestlé India Ltd – Comparable Company Valuation

## Overview

This project presents a relative valuation of Nestlé India Limited using Trading Comparable Analysis and Precedent Transaction Analysis.

The valuation has been prepared using publicly available financial information and recent FMCG sector transactions.

## Project Highlights

* Trading Comparable Analysis
* Precedent Transaction Analysis
* Enterprise Value calculations
* Valuation multiples
* Relative valuation summary
* Supporting calculations and assumptions
* Interactive dashboard
