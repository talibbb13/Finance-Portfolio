# Hindustan Unilever Ltd – Financial Model & DCF Valuation

## Overview

This project presents a fully integrated three-statement financial model and Discounted Cash Flow (DCF) valuation for Hindustan Unilever Limited (HUL).

The model has been developed using publicly available financial information and follows industry-standard financial modeling practices.

## Project Highlights

* Integrated Income Statement, Balance Sheet and Cash Flow Statement
* Dynamic assumption-driven forecasts
* Supporting schedules
* Discounted Cash Flow (DCF) valuation
* WACC and Terminal Value calculations
* Sensitivity analysis
* Interactive dashboard
* Model integrity and sanity checks
