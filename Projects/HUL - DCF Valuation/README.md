# Hindustan Unilever Ltd – Financial Model & DCF Valuation

## Overview

This project presents a fully integrated three-statement financial model and Discounted Cash Flow (DCF) valuation for Hindustan Unilever Limited.

The model was developed using publicly available financial information and follows standard financial modeling practices.

## Files

* Financial Model (.xlsx)
* Project Preview (.pdf)

## Model Features

* Historical financial statements
* Forecast assumptions
* Integrated Income Statement, Balance Sheet & Cash Flow Statement
* Supporting schedules
* DCF valuation
* Sensitivity analysis
* Model integrity checks
* Dashboard

## Valuation Method

* Unlevered Free Cash Flow (UFCF)
* WACC
* Terminal Growth Method
* Enterprise Value
* Equity Value
* Implied Share Price

* Financial Modeling
* Business Valuation
